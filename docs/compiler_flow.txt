│                           COMPILER USAGE                                  │
└─────────────────────────────────────────────────────────────────────────────┘

    Basic compilation:
    $ compiler.exe input.tl -o output.c

    Assembly output:
    $ compiler.exe input.tl -o output.s --asm

    Debug information:
    $ compiler.exe input.tl -o output.c --debug --verbose

    Module compilation:
    $ compiler.exe input.tl -o output.c --module-mode --module-dir ./modules

    Analysis flags:
    $ compiler.exe input.tl --tokens    # Show lexical analysis
    $ compiler.exe input.tl --ast       # Show abstract syntax tree
    $ compiler.exe input.tl --ir        # Show intermediate representation

    Multiple files:
    $ compiler.exe file1.tl file2.tl -o output.c

    Memory statistics:
    $ compiler.exe input.tl -o output.c --memory
